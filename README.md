# ToDoList API – RESTful Application

A lightweight REST API backend built with Node.js, Express, and PostgreSQL.
Designed for task management (To-Do lists) — a great base project for learning CRUD operations, database integration, and API design.

# Tech Stack

**Backend:** Node.js, Express

**Database:** PostgreSQL

**ORM/Client:** pg

**Environment:** dotenv, nodemon, cors

**Testing / Tools:** Postman, Docker (optional)

# Main Features

✅ Create new tasks
✅ Retrieve all tasks or a single task by ID
✅ Update task details (title, description, status)
✅ Delete tasks
✅ RESTful routes with proper error handling

# Technologies:

**Node.js + Express** – API server

**PostgreSQL** – database for task storage

**pg** – database communication library

**dotenv, cors, nodemon** – development and environment tools
